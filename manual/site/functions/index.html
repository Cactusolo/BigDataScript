Config function

Reading a config file can be done using config function which parses a file and returns `' pairs in a map. Parsing: Lines starting with '#' are ignored, so are blank lines. Name/Value delimiters can be any of ':', '=' or '\t' (the first one found in each line will be used). The following are valid and equivalent:

name : value
name = value
name \t value
