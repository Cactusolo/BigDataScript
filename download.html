1. Download & Install

Binary distributions are available for Linux and OS.X.

Download Linux 64bit Download OS.X.

Install

Uncompress the binary at your HOME directory 
$ cd 
$ tar -xvzf path/to/bds_linux.tgz

BDS's directoy .bds is 'hidden', since the name starts with a dot.

Add BDS's directory to your PATH 
echo "export PATH=\$PATH:\$HOME/.bds" >> $HOME/.bashrc

Requirements

BigDataScript requires Java 7.

3. Source code

The source code is available at GitHub, here we show how to compile and installe.

The source code is available on GitHub . Download the project as follows (you need to install Git first): 

# Clone repository
$ git clone https://github.com/pcingola/BigDataScript.git
BigDataScripts requires the following software:
  • Java (at least 1.6)
  • Go compiler (at least 1.0)
  • Ant (at least 1.7)
Once you downloaded the project, you can install: 
# Run 'install script
$ cd BigDataScript
$ ./scripts/install.sh
Then edit your .bashrc (or .bash_profile) and add the following lines 
# BigDataScript
export CLASSPATH=$CLASSPATH:$HOME/.bds/BigDataScript.jar
export PATH=$PATH:$HOME/.bds/
Optionally, if you have root access, you can do the following (to get bds available everywhere in your system 
$ su -
# cd /usr/bin/
# ln -s /home/your_user/.bds/bds
